# Travel Booking Dashboard

A practical frontend project built with React to practice real-world frontend engineering workflows, including authentication, protected routes, booking CRUD, form validation, local mock persistence, reusable UI components, and data fetching/mutation management using TanStack Query.

This project is part of my learning project:

**Frontend Engineer 2026 — Practical AI-Assisted Development**

The goal is not only to build a working app, but also to practice how modern frontend projects are structured, reviewed, tested, refactored, and documented.

---

## Overview

Travel Booking Dashboard is a mock travel booking management app. Users can log in, view dashboard data, browse bookings, search and filter bookings, view booking details, create new bookings, edit locally created bookings, and delete locally created bookings.

The app currently uses mock data and `localStorage` instead of a real backend. This allows the project to focus on frontend fundamentals and real-world frontend workflow before introducing backend integration.
